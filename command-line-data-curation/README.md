# CLI Data Entry Script

## Description
`cli_data_entry.py` is a command-line Python script that allows you to enter your favorite movie details such as title, release date, and box office earnings. 
The script displays some initial data using the Rich library and lets users add their own entries. 
After entering the data, it confirms each entry before saving it to a `movies_data.csv` file.

## Prerequisites
It uses the rich library to print styled text to the console, so you need to make sure you instal the rich library.

step 1 Ensure pip is installed:

python3 -m ensurepip --upgrade

step 2 Install the rich module using pip: 

python3 -m pip install rich

step 3 Run script: 

python3 cli_data_entry.py
